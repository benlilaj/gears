Tutorial: How Gearpad became offline-enabled

Gearpad is a web application that started its life as an ordinary web application, called HalfNote. The application allows a user to write and save a single note which is then available online.

Features

The features of HalfNote that are interesting to a web application designer, are:

  • sign-on/log-in
  • storage
  • sync

By adding Gears capabilities, the updated Gearpad application adds these features:

  • offline access
  • sign-on
  • sync
