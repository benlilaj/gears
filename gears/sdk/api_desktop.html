Desktop API [New in 0.3]

The Desktop module provides an interface for accessing desktop related functionality, such as creating shortcuts.

Example

<script type="text/javascript" src="gears_init.js"></script>
<script type="text/javascript">
var desktop = google.gears.factory.create('beta.desktop');
desktop.createShortcut("Test Application",
                       "An application at http://www.test.com/index.html",
                       "http://www.test.com/index.html",
                       [{"16x16": "http://www.test.com/icon16x16.png",
                         "32x32": "http://www.test.com/icon32x32.png",
                         "48x48": "http://www.test.com/icon48x48.png",
                         "128x128": "http://www.test.com/icon128x128.png"]});
</script>

Desktop class

Methods

createShortcut(name, description, url, icons)
Parameters: name is the name of the shortcut. It must be under 50 characters, and cannot contain any of the characters below: 
"\/:*?<>|

description is a text blurb that will be shown in the confirmation dialog.
url is the URL that will be launched when the user opens the shortcut.
icons is an object containing the URLs of up to four icons on properties named: 16x16, 32x32, 48x48, 128x128. The icons must be .png files with dimensions matching the field they are associated with. At least one of 16x16, 32x32, or 48x48 sizes must be specified.
Exceptions: Throws an exception if an error occurs.
Description: Attempts to create a desktop shortcut called 'name' that links to 'url' with an icon built from the images listed in the 'icons' object. A dialog is presented to the user with the text of 'description' and the icon with dimensions closest to 32x32. If the user confirms, a desktop shortcut is created.
