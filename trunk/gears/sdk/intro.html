Google Gears API Developer's Guide (Beta)

Introduction

Gears is an open source toolkit that enables developers to create web applications that can run offline.

Gears provides a browser extension that supports further modular extensions to the browser, using a JavaScript API. There are three Gears modules available: Database, WorkerPool, and LocalServer. An offline-enabled application may use one or more modules, depending on the needs of the application.

LocalServer Module

The LocalServer module is a specialized URL cache under application control.  There are two significant use cases:

  • Application Caching - caching of application resources that are required to load and run the application, including html, js, img, and css resources.
  • User Data Caching -  not intended for app resources, but for URLs managed by an app at runtime.
