What is Google Gears?

Google Gears is a browser extension that adds JavaScript APIs to those normally available to web applications. These APIs enable developers to create web applications that work offline.


Under what license is Google Gears available?

Google Gears is completely free (as in both beer and speech). Gears is open source and available under the New BSD license.


On what systems is Google Gears available?

Gears is available on Windows, Linux and Mac OS/X. It works with Firefox, Internet Explorer, with more browsers planned in the future.

Check out the Sample Applications to get an idea of how you can start using Gears today.


What are Gears modules?

Gears modules are units of functionality that can be used individually or together in your web application. The current modules are:

  • Database - Provides a full relational database, using SQLite. The Database module makes it possible to store large amounts of data locally. Full-text indexing is supported in multiple languages.
  • LocalServer - A specialized cache that stores and retrieves your application's resources. The LocalServer enables your application to start without an network connection, and to run offline.
  • WorkerPool - Makes it possible to run asynchronous processes, that can increase your application's performance, and keep the browser responsive during I/O-intensive operations.

What is the status of Gears?

Gears is currently a Developer Release and is not yet ready for large-scale consumer applications. We are eager to hear your feedback, and we will be updating the code as well as improving the API based on community input.

Visit the Google Gears Developer Forum to share your feedback.
